# adaptive-textview
    自动调整文本字号以符合其边界的TextView

# 使用
```groovy
    allprojects {
        repositories {
            ...
            maven { url 'https://jitpack.io' }
        }
    }
```

```groovy
    dependencies {
        implementation 'com.github.bravetou:adaptive-textview:1.0.0'
    }
```
